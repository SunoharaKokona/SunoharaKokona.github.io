6.成品效果

  • 左右 prompt：git 分支 / 执行时间 / Python 或 Node 版本一目了然
  • 命令高亮：正确绿色，错误红色，实时提示
  • 自动补全：按 → 接受灰色建议
  • 目录跳转：j down 直接到 ~/Downloads
  • CPU <1 %，启动延迟 <50 ms，颜值与性能兼得。
